1. Markdown 手册

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

本文简单介绍了 Markdown 的使用方法。

关于更多 Markdown 的使用方法，可以参考 Markdown 教程。

1.1. 标题

使用 # 表示标题，不同数量的 # 可以表示不同级别的标题，但最多 6 级。

1.2. 字体风格

使用 * 或 ** 表示斜体和粗体。

示例：这是斜体，这是粗体，这是加粗斜体，这是删除线。

1.3. 列表

使用 - 表示无序列表，使用 1. 表示有序列表。

  • 无序列表项

示例：

使用 `代码` 表示代码，用 ``` 使用表示代码块。

示例：
